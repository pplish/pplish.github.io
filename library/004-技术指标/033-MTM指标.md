# MTM
**动量指标**（Momentum Index，**MTM**）是一种专门研究股价波动的中短期技术分析工具。该指标的理论基础是价格与供求量的关系，认为股价的涨跌幅度随着时间的推移会逐渐变小，股价变化的速度和能量也会慢慢减缓，行情出现反转的可能性增大。
