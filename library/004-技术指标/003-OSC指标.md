# OSC
**震荡量指标**(Oscillator, **OSC**)也叫变动速率线，属于超买超卖类指标，是从移动平均线原理派生出来的一种分析指标。它反应的是当日收盘价与一段时间内平均收盘价的差离值，据此得出股价的震荡幅度。按照移动平均线原理，根据OSC的值可推断价格的趋势，如果远离平均线，就很可能向平均线回归。
