# PBX
**瀑布线指标**（**PBX**）是一种中线指标，一般用来研究股价的中期走势，可过滤掉盘中主力震仓洗盘或下跌行情中的小幅反弹，直观有效地反映大盘和个股的运动趋势。与普通均线系统相比，具有反应速度快，买卖点明确的特点，是一种在判断大势和个股股价运行趋势方面颇为有效的均线系统。
