# BWIDTH
**极限宽指标**（Boll Width，**BWIDTH**）是一种表示布林线开口大小的指标。布林线指标开口逐渐变小代表股价的涨跌幅度逐渐变小，多空双方力量趋于一致，股价将会选择方向突破。通常来说，BWIDTH指标值小于0.1时，股价发生突破的可能性较大，不过该值会随着股票不同而有差异，因此最好根据BWIDTH指标近一年的走势来确定其临界点。
