# VSTD
**成交量标准差指标**（Volume Standard Deviation，**VSTD**）是一种反应一定周期内成交量标准差的指标，可以有效的追踪成交量放大和缩小的趋势，尤其对股价顶部的研判较为准确。当股价已创出新高而该指标尚未创出新高的时候需保持谨慎，因为股价随时都有下跌的可能。
