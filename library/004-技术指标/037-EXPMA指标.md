# EXPMA
**指数平均线指标**（Exponential Moving Average，**EXPMA**）是一种趋势指标，用于判断价格未来的趋势。该指标由于计算的时候着重考虑了当天的价格，所以相对来说克服了其他指标信号滞后性的特点，是一种较好的趋势类中线指标。
