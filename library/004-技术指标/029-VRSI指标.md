# VRSI
**量相对强弱指标**（Volume Relative Strength Index，**VRSI**）通过反映股价变动的四个元素：上涨的天数、下跌的天数、成交量增加幅度、成交量减少幅度来研判量能的趋势，预测市场供求关系和买卖力道，是属于量能反趋向指标之一。与相对强弱指标RSI相比，计算方法和判断原理基本相同，但它重点考虑了量的因素，根据量比价先行的道理，能较好地超前判断价格走势。
