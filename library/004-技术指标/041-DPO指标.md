# DPO
**区间震荡线指标**（Detrended Price Oscillator，**DPO**）由Walt Bressert提出，是一种排除价格趋势的震荡指标。该指标在计算时扣除了前期移动平均价，因而消除了长期趋势对价格波动的干扰，更便于发现价格短期的波动和超买超卖水平。
