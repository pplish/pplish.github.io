# RSI
**相对强弱指标**（Relative Strength Index，**RSI**）由Welles Wilder提出，用于反映市场在一定时期内的景气程度。该指标理论认为，任何市价的大涨或大跌，均在0-100之间变动，根据常态分配的原理，RSI值应该多在30-70之间。RSI值高至80甚至90时认为市场已到达超买状态，市场价格会回落调整；低至30以下时认为是超卖状态，市价将出现反弹回升。
