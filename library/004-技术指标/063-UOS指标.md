# UOS
**终极波动指标**（Ultimate Oscillator，**UOS**）由Larry Williams提出，是一种经过优化的振荡指标。单一周期的振荡指标，对周期的选择较为敏感，不同市况下周期不同时，指标结果差异很大。为了改善这一状况，Larry Williams经过测试，提出了UOS指标，指标中包含三个周期参数，可以更好的适应不同的市况。
