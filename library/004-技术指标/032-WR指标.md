# WR
**威廉指标**（Williams Overbought/Oversold Index，**WR**）由Larry·Williams提出，是一种兼具超买超卖和强弱分界的指标。该指标通过分析收盘价在过去一段时间价格区间的相对位置，来度量市场处于超买还是超卖状态，主要作用在于辅助其他指标确认短期买卖信号。
