# ADV
**离散量指标**（Accumulation/Distribution Volume，**ADV**）通过统计成交量变动的趋势来推测股价趋势。ADVOL向上交叉其N日平均线时，视为买进讯号，向下交叉其N日平均线时，视为卖出讯号。
